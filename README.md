# APBD Task 3 - LINQ

University data analysis using LINQ in C#.

## How to Run

1. Clone the repo
2. Open in Rider
3. Make sure you have .NET 10
4. Run and pick a task number from the menu

## Structure

- Models - Student, Course, Lecturer, Enrollment
- Data - sample university data
- Exercises - all LINQ methods

## Tasks

16 LINQ tasks + 4 challenges covering where, select, orderby, join, groupby, aggregations and more.
